Google Chrome Mozilla Firefox

Расширения для YouTube

Enhancer for YouTube - лучшее расширение для кастомизации YouTube. Включает в себя смену темы, блокировку рекламы, скрытие элементов, расширение плеера, и множество прочих функций.

Google Chrome Mozilla Firefox

SponsorBlock — это огромная фан-база, которая вносит вклад в жизнь миллионов пользователей. Зрители на YouTube вручную ставят отметки на сегменты видео, которые будут автоматически пропускаться, будь то реклама, продакт-плейсмент или паузы.

Google Chrome Mozilla Firefox

Return YouTube Dislike - возвращает количество дизлайков на YouTube.

Google Chrome Mozilla Firefox

vidIQ — полезное расширение для блогеров, которое позволяет анализировать статистику чужих видео, отслеживать их рост, исследовать теги, тестировать превью и многое другое.

Google Chrome Mozilla Firefox

Бесплатные VPN сервисы

Browsec VPN — отличный и безопасный сервис, использующий шифрование данных. Среди бесплатных серверов доступны Нидерланды, Сингапур, США и Великобритания.

Google Chrome Mozilla Firefox

TouchVPN - один из популярнейших условно бесплатных VPN сервисов. Бесплатно доступна Россия, Нидерланды, США, и пара других стран.

Google Chrome Mozilla Firefox

Внешний вид

Dark Reader - полезное расширение для любителей тёмных тем. Расширение искуственно заменяет белые элементы на серые, создавая подобие тёмной темы там, где таковой нет. Зачастую может быть лишним, но сайты можно добавлять в исключения.

Google Chrome Mozilla Firefox
