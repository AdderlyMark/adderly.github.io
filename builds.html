Сборки от Аддерли

MakuOS Yuki (Windows 10 22H2, 19045.4894)


Удалённые приложения:

Paint 3D, 3D Viewer, Skype, Quick Assist, PDF Reader, Cortana, Feedback Hub, Get Help,
Microsoft Solitaire Collection, Mixed Reality Portal, Sticky Notes, Диктофон, Советы, Люди, Почта, Карты.

Применённые настройки:

Отключены рисунки и реклама в пуске
Отключена задержка контекстного меню
Отключена авто-перезагрузка при синем экране смерти
Отключено авто-добавление папок в панели быстрого доступа
Отключено авто-обновление системы
Отключено залипание горячих клавиш
Отключено сжатие обоев рабочего стола
Отключён SmartScreen
Включено отображение расширений файлов
Включено отображение скрытых файлов
Включён буфер обмена
Добавлена иконка "Мой Компьютер" на рабочем столе
В проводнике будет открываться "Этот ПК" вместо "Быстрый доступ"
На панели задач скрыта иконка "Просмотр задач" и "Новости и интересы"
При переносе и копировании файлов, диалог автоматически развёрнут
В старой панели управления по умолчанию выключены категории
Удалён Content Delivery Manager, который отвечает за предустановку рекламы
Таймер проверки дисков увеличен с 8 секунд до 1 минуты


Прочие изменения:

Интегрированны библиотеки Microsoft Visual C++ Redistributable 2008, 2010, 2012, 2013, 2015 - 2022.
Вместо предустановленных программ, в целях экономии места был помещён ярлык с ссылкой на - "Скачать программы".

Удалённые приложения:

Windows Mail, OneDrive, Windows Subsystem For Linux, WordPad, компоненты голосового ввода,
Eye Control, Map Control, Quick Assist, Retail Demo Content, Skype, Windows Mixed Reality,
PDF Reader, компоненты печати, Microsoft Edge Dev Kit.

В проводнике будет открываться "Этот ПК" вместо "Быстрый доступ"
